## Changer d'ordinateur
Le code est déjà sur GitHub. Sur n'importe quel ordi, il suffit de :
1. Ouvrir le Terminal
2. Taper : `git clone https://github.com/DufourBenedicte/DufourBenedicte.github.io.git`
3. Le dossier se télécharge complet, prêt à modifier

Pour Claude Code : lancer Claude Code dans ce dossier et le projet repart exactement là où on s'était arrêtés — Claude relit le code et reprend le contexte.
